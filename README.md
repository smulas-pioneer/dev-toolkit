# dev-toolkit
Development Toolkit


## Install
To install or update dev-toolkit, you can use the install script using cURL:

```
curl -o- https://raw.githubusercontent.com/vgmr/dev-toolkit/master/sync-toolkit/bin/dt-sync-toolkit | bash
```
